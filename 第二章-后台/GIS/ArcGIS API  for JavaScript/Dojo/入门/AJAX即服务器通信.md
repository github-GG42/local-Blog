### 三维数据成果格式之三(倾斜摄影模型`.slpk`文件)

- 是后缀名为`.slpk`的文件
- 包含`LOD`信息
- 有生产工具直接生成/`OSGB`格式转换
- 是开放的数据格式,符合**i3S**标准(`OGC`的三维数据标准)
- ![1570975535381](D:\总结\img\1570975535381.png)

### 大一统的`i3S`标准

#### 支持类型:

- 倾斜摄影模型数据
- 多面体图层数据
- `BIM`模型数据
- 点云数据
- 点数据
- 线数据(4.12暂不支持)
- 面数据(4.12暂不支撑)

#### 通过**`ArcGIS Pro`**工具转为符合`I3S`标准的`SLPK`文件

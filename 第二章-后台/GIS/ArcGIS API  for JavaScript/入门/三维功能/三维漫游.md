# 三维漫游

`view.goTo(target,options?)`

## 说明:

1. `target`:目标位置

2. `option`:可选参数

   包含:

   `animate`:运动模式

   `speedFactor`:运行速度;值为:**0.0~1.0**,值越小,运行速度越慢

   `duration`:运行时间

   `maxDuration`:最大时长

   `easing`:运行方式,类型为一个函数,默认会接收一个参数,该参数为运行的总时长;自定义该函数,可以实现自定义运行方式

作用:控制从当前视角跳转到目标位置运行时长和运行速度.
